# [Productos CRUD]


## Descripcion del Sistema:
 "Aplicacion web de comercio electronico para una tienda que permite explorar productos, gestionar un carrito de compras y persistir los datos."

---

## Tecnologias Utilizadas

El proyecto fue desarrollado utilizando el siguiente stack tecnologico:

* **Frontend / UI:** [Ej: HTML5, CSS3, JavaScript (ES6+), Bootstrap]
* **Entorno de Ejecucion:** [Node.js]
* **Gestor de Paquetes:** [npm]
* **Herramientas de Construccion:** [Vite]
* **Base de Datos (Si aplica):** [MySQL]

---

## Instrucciones de Instalacion

Segui estos pasos para configurar el entorno de desarrollo local:

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/tu-usuario/tu-repositorio.git](https://github.com/roccoch/tp-productos)
    cd nombre-del-proyecto
    ```

2.  **Instalar las dependencias:**
    Ejecuta el comando correspondiente segun el gestor de paquetes que utilices:
    ```bash
    npm install
    ```

3.  **Configurar las variables de entorno:**
    * Crea un archivo `.env` en la raiz del proyecto (basandote en `.env.example` si existe).
    * Define las variables necesarias (puertos, claves de API, conexiones a bases de datos, etc.).
